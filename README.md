## Descripción

Este proyecto consiste en una aplicación desarrollada en Python para la gestión de datos de países.

El sistema permite:

* Agregar países.
* Actualizar información.
* Buscar países.
* Filtrar por continente.
* Filtrar por población.
* Filtrar por superficie.
* Ordenar países.
* Obtener estadísticas.

Los datos se almacenan utilizando un archivo CSV y se gestionan mediante listas y diccionarios.

## Requisitos

* Python 3.x

Instalación opcional:

pip install colorama

## Ejecución

Ejecutar:

python Trabajo_integrador.py
